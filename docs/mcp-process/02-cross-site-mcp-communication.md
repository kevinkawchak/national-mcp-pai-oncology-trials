# 02 — Cross-Site MCP Communication

**National MCP-PAI Oncology Trials Standard v0.9.0**

Detailed diagram of MCP server interactions between clinical trial sites, sponsors,
CROs, and the federated coordination layer for multi-site oncology trials.
