# 03 — Clinical System Integration

**National MCP-PAI Oncology Trials Standard v0.9.0**

FHIR R4, DICOM, and identity system integration pathways showing how MCP servers
connect to real hospital infrastructure through standardized adapters.

## Terminology Mapping Integration

```
+-----------------------------------------------------------------------------+
|                     TERMINOLOGY MAPPING INTEGRATION                         |
+-----------------------------------------------------------------------------+
|                                                                              |
|  SOURCE DATA            TERMINOLOGY SERVICE            NORMALIZED DATA      |
|  +------------------+   +---------------------+        +------------------+ |
|  |                  |   |                     |        |                  | |
|  | ICD-10-CM Code   +-->+ Map to Standard     +------->+ Unified Code    | |
|  | (e.g., C34.90)   |   | Code System         |        | Representation  | |
|  +------------------+   |                     |        +------------------+ |
|  |                  |   | +---------------+   |        |                  | |
|  | SNOMED CT Code   +-->+ | ICD-10 <-> CT |   +------->+ Cross-system    | |
|  | (e.g., 254637007)|   | +---------------+   |        | Lookup Table    | |
|  +------------------+   | | SNOMED <->    |   |        +------------------+ |
|  |                  |   | | LOINC         |   |        |                  | |
|  | LOINC Code       +-->+ +---------------+   +------->+ Standardized    | |
|  | (e.g., 21889-1)  |   | | RxNorm <->   |   |        | Observations    | |
|  +------------------+   | | NDC           |   |        +------------------+ |
|  |                  |   | +---------------+   |        |                  | |
|  | RxNorm Code      +-->+                     +------->+ Normalized      | |
|  | (e.g., 583214)   |   |                     |        | Medications     | |
|  +------------------+   +---------------------+        +------------------+ |
|                                                                              |
|  Oncology-Relevant Code Systems:                                            |
|  ICD-10-CM  : C34 (Lung), C50 (Breast), C61 (Prostate), C71 (Brain)       |
|  SNOMED CT  : 254637007 (NSCLC), 93880001 (Adenocarcinoma)                 |
|  LOINC      : 21889-1 (Tumor size), 85319-2 (TMB)                         |
|  RxNorm     : 583214 (Pembrolizumab), 1946825 (Osimertinib)               |
|                                                                              |
+------------------------------------------------------------------------------+
```
